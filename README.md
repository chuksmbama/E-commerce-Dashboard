# E-commerce-Dashboard
## Project Overview 
The dashboard provides a high-level overview of key business metrics including:
- revenue: $30M
- Profit: $17M
- Products sold: 713k (Thousand)
- Profit by zone: Comparison across four regional zones
- Profit and products sold by year: Trends from 2015-2020
- Customer Demographics: Gender distribution (Male 48.41%, Female 51.59%)
- Customer Rating: satisfaction levels from dissatisfied to very satisfied
- Order Status: Delivered vs Returned rates


##  Key Features
- Dynamic filtering by product category
- Custom KPIs and DAX measures
- Year-over-year profit and sales comparison
- Engaging data visuals for insightful storytelling
- interactive and user-friendly dasboard design

## Tool and Technology
- Power BI for data visualization
- DAX for measure calculations
- Excel/CSV as data source
- Power query for data cleaning and transformation


## Insights Gained 
- Profit peaked in 2020 ($5.7M) indicating strong revenue growth
- Zone 3 was the most profitable region ($7.2M)
- Customer satisfaction was balanced across rating levels (23% per group)
- Delivered orders made up 73% of total orders
