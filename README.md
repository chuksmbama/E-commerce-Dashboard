# E-commerce-Dashboard
## Project Overview 
This dashboard provides information about the e-commerce company's performance from 2015 to 2020, including revenue, profit, total products sold, customer demographics, and customer distribution throughout Ghana. 


## Tool and Technology
- Power BI for data visualization
- DAX for measure calculations
- Excel/CSV as data source
- Power query for data cleaning and transformation
